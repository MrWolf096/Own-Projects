This  Library can : - create a library in the data base;
                    - help the user to insert new books; 
                    - help the user to insert new authors, new publishing house , new reviews and stars to the book that are created;
                    - shows the books that have been created;
                    - shows a specific author by name and surname;
                    - shows the age of the authors that helped created a specific book;
                    - shows the oldest book in the library;
                    - delete a specify publishing house;
                    - check if the book is able to be borrower;
                    - connect the author or authors to a book;
                    - create an account of a client that wants to borrow a book from the library;
                    - borrow a book if the client has a account in the library;
